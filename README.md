This project implements a working digital dice circuit built on a printed circuit board (PCB). Electronic components were soldered onto the board to create a device that simulates a dice roll and displays the result using LEDs.
The project involved:
Assembling and soldering components onto a PCB
Understanding digital circuit behavior
Testing and debugging hardware connections
Implementing a circuit that generates random dice values
This project demonstrates practical experience with basic electronics, PCB assembly, and hardware debugging.
